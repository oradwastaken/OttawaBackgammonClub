# Ottawa Area Backgammon Club Dashboard

Live site: https://oradwastaken.github.io/OttawaBackgammonClub/

## Updating the Data

1. Open the dashboard
2. Click the clipboard icon (top-right)
3. Drag in your latest Excel file
4. Click **Apply Data** — verify the stats look right
5. Click **Publish**
6. Done — the site updates in about a minute

## First-Time Setup

This only needs to be done once per browser:

1. On the dashboard, click the clipboard icon
2. Click **Settings** at the bottom
3. Enter your repository name (e.g. `oradwastaken/OttawaBackgammonClub`)
4. Enter your Access Token
5. Click **Save Settings**

If you need a new token: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Give it **Contents: Read and write** permission for this repository.
